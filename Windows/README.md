# Windows

模拟Linux环境

- git-for-windows
- [cygin](http://x.cygwin.com/)

> tasklist 
- 相当于 ps 命令, 查看帮助为 `tasklist /?`
- 按可执行文件查找运行的进程 `TASKLIST  /FI "IMAGENAME eq idea64.exe"`

- 资源管理器插件, 支持多标签, 类似于Chrome的使用 [clover](http://cn.ejie.me/)
